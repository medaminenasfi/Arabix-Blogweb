# Arabix Blog Web

A modern Arabic-language blog landing page built with Next.js 16 and Tailwind CSS. It showcases featured articles, category highlights, newsletter signup, and CTAs tailored for content publishers targeting MENA audiences.

## Tech stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript + React 19
- **Styling:** Tailwind CSS 4 with custom utility classes
- **UI components:** Radix UI primitives + shadcn-inspired component library in `components/ui`
- **Animation hooks:** `use-scroll-animation` for subtle reveal effects
- **Icons & charts:** Lucide, Recharts support (via shared UI kit)

## Key sections

1. **Header** with logo, navigation, search, and theme toggle.
2. **Hero** featuring a highlighted story, metrics, and CTA buttons in Arabic.
3. **Latest articles** grid rendered from reusable `ArticleCard` components.
4. **Categories** list with animated tags and icons.
5. **Newsletter** subscription block connected to a form component.
6. **CTA + Footer** encouraging readers to explore more and follow social channels.
